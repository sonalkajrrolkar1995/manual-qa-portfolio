# Manual QA Portfolio

QA deliverables covering three applications: SauceDemo (web e-commerce), ReqRes (REST API mock), and Restful Booker (booking REST API). All analysis is based on actual testing sessions.

This is not a tutorial and not academic content. It is the kind of documentation a QA engineer produces during a real testing engagement - risk analysis, test design decisions, exploratory session notes, defect reports, and system-level thinking.
